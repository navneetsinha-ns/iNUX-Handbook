# Best Practice Examples for Implementation in Education

This directory contains two reports documenting **best practice examples for the implementation of interactive digital learning tools in hydrogeology education**, developed within the iNUX project.

The reports focus on **didactic integration, practical effort, and lessons learned** from real teaching scenarios. Rather than describing the technical details of the tools themselves, the emphasis is placed on how interactive materials were embedded into lectures, exercises, and courses, and how they supported conceptual understanding and learning outcomes.

Together, the reports illustrate transferable practices for using interactive tools in both **Basic** and **Applied Hydrogeology** education.

## Scope of the Reports

The best practice examples address:

- Integration of interactive tools into existing courses and curricula
- Didactic design and teaching workflows
- Required preparation effort and technical setup
- Observed learning outcomes and student feedback
- Lessons learned and recommendations for first-time adopters

The target audience includes **university teachers, course coordinators, and educational developers** interested in adopting interactive educational tools in undergraduate, graduate, or professional training contexts.
